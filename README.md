# 📄 个人简历在线生成器 (Resume Generator)

这是一个基于 HTML/CSS/JS 的轻量级个人简历生成器。它不仅是一份静态简历，更是一个功能强大的在线编辑工具。

🔗 **在线演示**：[点击这里查看](https://zilingcunzhang.github.io/) (假设这是你的 GitHub Pages 地址)

## ✨ 核心特性

- **🎨 多主题切换**：内置 4 款专业配色（稳重绿、商务蓝、优雅紫、极简灰），一键换肤。
- **✍️ 所见即所得编辑**：点击页面上的任何文字即可直接修改，支持从 Word/网页纯文本粘贴（自动清除格式）。
- **💾 自动保存**：基于浏览器 `LocalStorage` 技术，编辑内容实时保存到本地，刷新页面或关闭浏览器数据不丢失。
- **🖨️ 完美 A4 打印**：专为 A4 纸张优化的 CSS 打印样式，自动处理分页、页眉和边距，支持导出为标准 PDF。
- **🖼️ 头像更换**：支持本地上传头像（数据仅保存在本地浏览器）。
- **🔒 隐私安全**：所有数据均存储在您的本地浏览器中，不会上传到任何服务器，绝对安全。
- **📱 响应式设计**：完美适配手机、平板和桌面端阅读。

## 🚀 如何使用

### 方法一：直接使用 (推荐)
1. Fork 本仓库。
2. 开启 GitHub Pages。
3. 打开你的 GitHub Pages 链接，直接在网页上修改内容。
4. 点击底部的 **"导出 / 打印简历"** 按钮保存为 PDF。

### 方法二：本地运行
1. 克隆仓库到本地：
   ```bash
   git clone https://github.com/zilingcunzhang/zilingcunzhang.github.io.git
   ```
2. 直接用浏览器打开 `index.html` 文件。
3. 开始编辑你的简历。

## 🛠️ 技术栈

- **HTML5**: 语义化标签结构
- **CSS3**: CSS Variables (主题切换), Flexbox/Grid 布局, @media print (打印适配)
- **JavaScript (ES6+)**: LocalStorage 数据持久化, 交互逻辑

## 📝 打印/导出 PDF 技巧

1. 点击页面底部的 **"导出 / 打印简历"** 按钮。
2. 在打印设置中：
   - **目标打印机**：选择 "另存为 PDF"。
   - **纸张尺寸**：选择 "A4"。
   - **边距**：选择 "无" 或 "最小"。
   - **选项**：务必勾选 **"背景图形"** (否则主题色块无法显示)。
